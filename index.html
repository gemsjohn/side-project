Git

    Tool used to help developers manage code.

    1. Cloud.

    2. Collaborate.

    3. Version control.


    Add: git add [ file or -A ]

    Change Directory: cd folder

    Clone Repository: git clone git@github.com:username/repository

    Commit File: git commit -m "comment"

    File Creation: touch file_name

    Folder Creation: mkdir folder_name

    New Repository: git init

    Push to 'main' branch: git push origin main

    Status: git status
